## Week 1 : Python and Data Science Fundamentals

### Summary
During the first week, I learned the main tools and concepts needed for data analysis using Python. I started by setting up the development environment and then practiced Python fundamentals, NumPy, Pandas, and Matplotlib.

I applied these concepts to vehicle datasets by performing numerical calculations, cleaning and filtering data, creating summaries, and visualizing the results using different types of plots.

### Topics Covered
- Environment setup and package installation.
- Python functions, list comprehensions, and basic OOP.
- NumPy arrays, indexing, broadcasting, and statistical operations.
- Loading, cleaning, filtering, and grouping data using Pandas.
- Creating line, scatter, bar, and histogram plots using Matplotlib.
- Combining NumPy, Pandas, and Matplotlib in one mini-notebook.
